## Whispur v0.1.0

First public release of Whispur — an open-source macOS menu-bar voice dictation app.

### Features
- Menu-bar app with recording overlay
- Dual shortcut triggers: hold-to-talk and toggle
- Apple Speech-to-Text by default (no API key needed)
- Pluggable STT providers: OpenAI, Deepgram, ElevenLabs
- Optional LLM post-processing: OpenAI, Anthropic, Groq
- Prompt presets and run log
- Card-based Settings with Setup checklist

### Install
Download `Whispur.dmg`, open it, and drag Whispur into Applications.

Signed and notarized by Apple — no right-click-to-open workaround needed.

### Requirements
- macOS 14+
- Microphone and Accessibility permissions
